## Notes for the ingest pipeline

- **Industry Dive shared sender:** Banking Dive, Payments Dive, and other Dive newsletters all originate from `industrydive.com` infrastructure with per-publication `From:` names. Classifier should key on `List-ID` / `Sender` headers rather than just the From-domain when routing.
- **Substack shared sender:** Mikula, Taylor (also has fintechbrainfood.com), Johnson, Rubinstein, Sokolin, Boboev, Gustafson, Davis all send via `substack.com` (typically `newsletter+<slug>@substack.com` or DKIM-aligned with the custom domain). Use the `List-ID: <slug>.substack.com` header for routing — it's stable.
- **CoinDesk:** Both The Node and Crypto for Advisors send from coindesk.com; differentiate via List-ID or campaign params.
- **Forwarding:** All the above are plain HTML and forward cleanly to AgentMail. Bloomberg Money Stuff occasionally inlines images from `bloomberg.net` CDN — fine for HTML parse, but the classifier should tolerate large `<img>` payload.
- **Paid tiers flagged:** Net Interest, Fintech Business Weekly, Tearsheet TS Pro, Fintech Blueprint, Mostly Metrics, Fintech Takes all have meaningful paid content. Free tier is enough to start; upgrade Mikula + Rubinstein first if budget exists.
- **Not subscribed / dropped from request:**
  - **The Fintech Nest** — could not verify; not currently active under that name. Substituted **Fintech Blueprint** (Lex Sokolin).
  - **Petal & Pixel** — no community-banking newsletter found under that name. Substituted **Independent Banker** (ICBA) and **The Bank Slate** (Paul Davis) for the community-bank slot.
  - **Cap Hill Crypto** — last edition August 2025, looks dormant; replaced its policy slot with **Blockworks Forward Guidance** (inside the Blockworks Daily bundle) and **Stablecoin Insider**.
